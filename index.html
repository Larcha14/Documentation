      Примеры кода

      Реализация функции cola_let

      cola_let x' x = x + 1

      x' 5

      Вывод программы:

      6

      Реализация факториала

      cola_let cola_rec factorial n = 
                          cola_if n <= 1 cola_then 1 
                    cola_else n * factorial (n - 1)

      cola_let result = factorial 5

      Вывод программы:

      120

      Реализация списков

      cola_let result = cola_head [ cola_sin 0; cola_cos 0]

      Вывод программы:

      0
